## 使い方

1. ホームで科目別の着手率・正答率・未克服誤答を確認
2. 追加科目はまず「Aランク優先」または「弱点優先」20問
3. 財務・会計は、CVP・経営分析・NPV・CFを優先
4. 企業経営理論は、戦略・組織・マーケティングの言い換え問題を優先
5. 運営管理は、在庫・品質・IE・店舗管理を優先
6. 最後は誤答のみ復習で2回連続正解にする
