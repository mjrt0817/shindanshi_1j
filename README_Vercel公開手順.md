## GitHub / Vercel

リポジトリ直下に以下を置いてください。

```text
index.html
firebase.rules
vercel.json
README_Vercel公開手順.md
```

Vercelの設定は、Framework Preset は Other、Build Command と Output Directory は空欄で問題ありません。

## Firebase

Googleアカウント同期を使う場合は、Firebase Authentication の Google を有効化し、Vercel のドメインを Authorized domains に追加してください。
Firestore Rules は `firebase.rules` を貼り付けてください。

## 80点チャレンジの使い方

1. 科目別の着手率を85%以上に近づける
2. 未克服誤答を5問以下にする
3. 「80点チャレンジ」で応用・横断論点を解く
4. 最後は誤答のみ復習で2回連続正解にする
